Tags Vs Elements Vs Attributes

Tag

* it is used to insert documents, folders, images, or files that enhance the look and feel of content or perform an actions

* each tag is enclosed between two angle brackets(usually less than and greather than)

* 90% tags are come with start tag and end tag

for example => h1, p, body, head, title etc.

* few tag are self closing / void tag / empty tag. for e.g. br, hr, img, etc.

Element

* An HTML file is made of elements

* these element are responsible for creating web pages and define content in that webpage.

* An element usually consist of a start tag , close tag and content inserted between them.

* An element is a collection of start tag, Attributes, end tag, content between them.

Attribute

* Attribute provide additional information about element

* Attribute are always specified in the start tag

* Attribute usually come in name/value pairs like :- name="value"

to change behaviour of an element we use Attribute.
