What is HTTP / HTTPS

HTTP >An HTTP stands for Hypertext Transfer Protocol. The HTTP protocol provides communication between different communication systems. when the user makes an HTTP request on the browser, then the webserver sends the requested data to the user in the form of web pages. in short we can say that the HTTP protocol allows us to transfer the data from the server to the client.
HTTPs > HTTPs stands for Hypertext Transfer Protocol Secure . It is highly advanced and secure version of HTTP. It alows the secure transaction by encryptingthe entire communication with SSL. It is a combination of SSL/TLS (sockets layer/transport layer security)protocol and HTTP. it provides encrypted and secure identification of a network server.

What is client server

Client server is a relationship in which one program, the client requests a service or resource from another program, the server. The label client server was previously used to distinguish distributed computing by PCs from the monolithic, centralized computing model used by mainframes. today computer transactions in which the server fulfills a request made by a client are very common. The client server model has become one of the central ideas of the client establishes a connection to the server over a LAN or WAN such as the internet.

What is Domain

A domain is any collection of nodes, workstations, devices, file servers, database servers, etc. which are meant to be sharing the resources and information with with each other efficiently. Further we have various different kinds of subdomains.

What is Host

A device like a laptop, computer that connects to the network is a host. A networkis made up of multiple devices and each device will have an IP address that will help identify that particuler device in the network. When one host need to communicate with another device, the data reaches the correct designation using the IP address of the device with which it wishes to communicate. At times, hosts use hostname instead of IP addresses to communicate over the network in which case the DNS will convert the hostname to an IP address.

What is FTP

File Tranfer Protocol is a standard network protocol used for the transfer of files from one host to another over a TCP (Transmission Control Protocol)
based network, such as internet. FTP works by opening two connections that link the computer trying to communicate with each other. one connection is designated for the commands and replies that get sent between the two client, and other channel handles the transfer of data.

What is path

A path is a string of characters used to uniquely identify a location in a directory structure. It is composed by following the directory tree hierarchy
in which components, separated by a delimiting character, represent each directory.

What is extension

An extension in computer operating systems, is a piece of software that enhances or "extends" the capabilities of a programming language or other applications. An extension adds extra features to an already working standalone application. Usually an extension itself is not independently functional and needs other software with which to work.

What is URL-URI/URN

URL > URL stands for Uniform Resource Location . URL is a subset of URL that describes the network address or location where the source is available. URL beings with the name of the protocolto be used for accessing the resource and then a specific resource location. URLs build on the Domain Name Service(DNS)to address hosts symbolically and use a file-path like syntex to identify specific resource at a given host.
~URN > URNstands for Uniform Resource Name. It is a URI that uses a URN scheme. URN does not imply the availability of the identified resource. URNs are located-independent resource identifiers and are designed to make it easy to map other namespaces into URN space.

What is Address bar

An address bar is a component of an internet browser which is used to input and show the address of a website. The address bar helps the user in navigation by allowing entry of an internet protocol address or the uniform resource locator of a website. It can also save previously used addresses for future reference.

What is Title bar

Title bars are horizontal bars at the top of a window in a graphical user interface (GUI). The title of the software, the name of the current document of file, or other information that identifies the content of the window is shown in the title bar of the window. The titlebar may also have other characteristics that rely on the operating system and the software that displays the title bar.
